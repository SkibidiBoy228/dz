PUT приклад

PUT неможливо зробити з браузера напряму (forms не підтримують PUT), але можна зробити через JavaScript:

fetch('/http-help/', {
    method: 'PUT',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({"action": "update", "value": 123})
});

PATCH приклад

fetch('/http-help/', {
    method: 'PATCH',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({"field": "name", "value": "NewName"})
});

DELETE приклад

fetch('/http-help/', {
    method: 'DELETE'
});
